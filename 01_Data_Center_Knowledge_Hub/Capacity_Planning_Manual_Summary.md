# Capacity Planning Manual - Summary

## Purpose

The capacity planning process converts vessel bookings, discharge information, BAPLIE files and operational constraints into clear dashboards for decision-making.

## What it supports

- Booking versus vessel capacity.
- TEU, weight and plug visibility.
- Shortfall and overbooking discussions.
- Cut decisions before closing.
- Commercial follow-up before deadlines.
- Operational alignment across departments.

## Business value

The manual process previously required several files, repeated checks and manual consolidation. The automated process reduces that work into a repeatable flow: input data, Python processing, HTML report and management review.

## Data migration relevance

This process is a strong candidate for centralization because it has clear inputs, clear transformations and clear decision outputs.
