# Maritime Operations Intelligence Portfolio

Built to replace manual vessel planning workflows with decision systems.

This portfolio is not a catalog of dashboards. It is a public, sanitized case study of how operational data can be turned into earlier capacity decisions, shortfall visibility, and execution control across multi-port container shipping operations.

## Impact

- Reduced vessel planning cycle time from 4h to 20 min per vessel/day.
- Processed 950+ container moves in under 20 minutes.
- Identified shortfall risk before execution, including 99+ TEUs at port level.
- Supported multi-port operations across BRSSA, BRFOR, BRRIO, and BRIOA.
- Connected booking control, capacity planning, gate-in visibility, final moves, and bayplan views into one decision flow.

## Main Case

**Capacity and Shortfall Intelligence**

This case prevented capacity planning errors before vessel execution by identifying port-level shortfall and exposing the gap between planned demand, operational capacity, and execution status.

The core operational question was simple:

> Do we have enough vessel capacity by port, equipment type, TEU, weight, and execution window before the operation starts?

The system answered that question faster than manual spreadsheet review and made the risk visible before the vessel plan became operationally expensive to correct.

## What This Portfolio Shows

End-to-end operational intelligence for container shipping:

- Booking control and commercial demand visibility.
- Capacity planning by TEU, weight, port, equipment type, reefer, NOR, IMO, and OOG.
- Shortfall analysis before vessel execution.
- Gate-in visibility and load readiness.
- Final moves tracking for execution follow-up.
- 3D bayplan visualization for operational context.
- HTML reports designed for fast review by planning, commercial, and execution teams.

## Why It Matters

Without this system:

- planning decisions are delayed;
- shortfall is identified too late;
- teams compare different versions of the same operation;
- execution runs with blind spots;
- senior review depends on manual spreadsheet consolidation.

With this system:

- decisions happen before execution;
- capacity risks are visible early;
- planning, commercial, and operations read the same numbers;
- exceptions are easier to prioritize;
- vessel planning becomes repeatable and auditable.
